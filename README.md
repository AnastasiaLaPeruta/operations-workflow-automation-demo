
# Operations Workflow Automation Demo

## Overview
This project demonstrates a simple AI/data automation workflow for operations-heavy businesses.

The workflow:
1. Cleans messy intake spreadsheet data
2. Standardizes names/emails/statuses
3. Removes duplicates
4. Generates a revenue summary report

## Technologies Used
- Python
- Pandas

## Files
- messy_intake_data.csv
- automation_demo.py
- cleaned_data.csv (generated after running)
- revenue_summary.csv (generated after running)

## Example Use Cases
- Insurance intake cleanup
- Healthcare operations reporting
- Legal client intake normalization
- Recruiting pipeline cleanup

## Value
This type of workflow can reduce manual spreadsheet work and improve reporting accuracy.
